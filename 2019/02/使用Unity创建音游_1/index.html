部分代码：

因为音游的key，是先下落的，在声音播放时，key才落到指定位置（所以不同项目的delay都不相同，需要后期调试）这里提供了一个鼓的声音，方便调试

public AudioSource F;
public AudioSource B;
public float delay;
public IEnumerator PlayMusic()
    {
        //播放
        B.PlayOneShot(b);
        yield return new WaitForSeconds(delay);
        F.PlayOneShot(f);
    }
